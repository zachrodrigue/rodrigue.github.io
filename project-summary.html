Capstone Project: Bellabeat

Projet de fin d'études : Bellabeat

Conducted an Exploratory Data Analysis (EDA) on fitness data from non-Bellabeat devices to identify user trends, informing Bellabeat's product development and marketing strategies to enhance customer engagement.

J'ai réalisé une Analyse Exploratoire des Données (EDA) sur analysé les données de fitness provenant d'appareils non Bellabeat pour identifier les tendances des utilisateurs, informant ainsi le développement de produits et les stratégies marketing de Bellabeat afin d'améliorer l'engagement des clients.

Tools used: R Programming, ggplot2, tidyverse, R Markdown

Outils utilisés : Programmation R, ggplot2, tidyverse, R Markdown
